🌺 BhaktiApp – A Java Console-Based Spiritual Playlist

🙏 Welcome to **BhaktiApp**, a Java-based console application that brings peace, devotion, and powerful mantras to your fingertips.

This app lets users select and listen to sacred paths of Bhakti such as:
- Lord Ganesh Stotram
- Ganapati Atharvashirsha
- Sadashiv Ashtakam
- Krishnaashtakam
- Surya Mantra

Each choice offers a beautiful mantra and a YouTube link for listening — creating a simple yet soulful spiritual console experience.

---

## 💻 Technologies Used
- **Core Java**
- Java Control Structures (`switch`, `Scanner`)
- Console-Based Input/Output

---

## ✨ Features
- Simple, beginner-friendly console interface
- Switch-based user choice logic
- Spiritual quotes and real mantra references
- Carefully curated YouTube links for each mantra
- Emotional and devotional touch in code styling ❤️
